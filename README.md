# monitoring-volume-solar-esp32
Monitoring Volume Solar berbasis IoT menggunakan ESP32, sensor ultrasonik JSN-SR04T, dan modul Ethernet LAN8720. ESP32 bertindak sebagai HTTP server dengan endpoint JSON. Dashboard web (HTML, CSS, JS) di-host via LittleFS ESP32 untuk monitoring volume, tinggi, dan persentase solar, serta bisa mengubah konfigurasi IP statis ESP32 dalam jaringan LAN.
